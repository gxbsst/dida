# 滴答清单(TickTick)

> Task Manager: [滴答清单](https://dida365.com/) && [TickTick](https://ticktick.com/)

dida: Easily add task to 滴答清单(TickTick) by shortcut.

<img src="screenshot.png" width="600">

## Dev

```
$ npm i && cd app && npm i
```

### Run

```
$ npm start
```

## Build

```
$ npm run build:dida
```
or
```
$ npm run build:tick
```
